# Project Structure

```
Possah_1.0/
│
├── app/                        Next.js App Router
│   ├── (shop)/                 All public storefront pages
│   │   ├── page.tsx            Homepage
│   │   ├── shop/[category]/    Category listing + PDP
│   │   ├── best-sellers/       Best sellers page (is_top_selling)

│   │   ├── bridal/             Bridal editorial (is_bridal flag)
│   │   ├── women/              Women landing page
│   │   ├── new-in/             New arrivals listing
│   │   └── ...                 cart, checkout, account, wishlist, etc.
│   ├── admin/                  Admin dashboard (auth-protected)
│   │   ├── products/           Product CRUD + ProductForm
│   │   ├── orders/             Order management + fulfilment
│   │   ├── categories/         Category manager
│   │   ├── homepage/           Homepage config editor
│   │   └── ...                 coupons, reviews, journal, media, settings
│   ├── api/
│   │   ├── admin/              Admin API routes (service role)
│   │   ├── products/           Public products API (paginated listing)

│   │   └── payments/           Razorpay verify + webhook

│
├── components/
│   ├── layout/                 Header, Footer, MobileNav, AnnouncementBar

│   ├── homepage/               HeroSlider, OccasionGrid, CategoryCircles, etc.
│   ├── pdp/                    ProductGallery, ProductInfo, CraftBehind, etc.

│
├── lib/
│   ├── supabase/               client, server, admin, public clients

│   ├── env-setup.md            Environment variables reference
│   ├── webhook-setup.md        Razorpay webhook setup
│   └── archive/                Historical plans and build guides (read-only)
│
├── tests/
```

## Key files — the ones you'll touch most

| File | When you touch it |
|---|---|
| `components/layout/Header.tsx` | Nav changes |
| `components/shop/FilterSidebar.tsx` | Occasion / fabric / size filter changes |
| `lib/validations/admin-products.ts` | Adding product fields |
| `app/admin/products/ProductForm.tsx` | Admin product form changes |
| `scripts/data_ops/lib/products.mjs` | Catalog changes (before pipeline re-run) |
| `scripts/data_ops/category_map.json` | After adding categories |
| `supabase/migrations/` | Any DB schema change |
| `docs/supabase-sql-actions.md` | Any SQL to run in production |
